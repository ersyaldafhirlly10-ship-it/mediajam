# 🕐 Media Pembelajaran: Membaca Jam

Media pembelajaran interaktif berbasis web untuk membantu siswa SD belajar membaca jam analog secara mandiri dan menyenangkan.

---

## Deskripsi

File HTML ini menampilkan jam analog yang bergerak mengikuti waktu nyata (real-time), dilengkapi dengan fitur kuis dan panduan cara membaca jam. Tidak memerlukan instalasi atau koneksi internet — cukup buka di browser.

---

## Fitur

- **Jam analog real-time** — jarum bergerak otomatis mengikuti waktu perangkat
- **Jam digital** — ditampilkan di bawah jam analog untuk membantu mencocokkan bacaan
- **Keterangan jarum** — penjelasan jarum pendek (jam), panjang (menit), dan merah (detik)
- **Kuis interaktif** — siswa menebak jam terlebih dahulu sebelum melihat jawaban
- **Langkah membaca jam** — panduan 3 langkah mudah dengan contoh
- **Tips menit × 5** — pengingat rumus membaca menit dari angka jarum panjang

---

## Cara Penggunaan

### Membuka File
1. Unduh file `media_belajar_jam.html`
2. Klik dua kali file tersebut, atau buka dengan browser (Chrome, Firefox, Edge, Safari)
3. Tidak diperlukan koneksi internet

### Di Kelas
1. Tampilkan file di proyektor atau layar
2. Minta siswa menutup/tidak melihat jam digital di bawah
3. Tanyakan: *"Sekarang pukul berapa?"*
4. Setelah siswa menjawab, klik tombol **Lihat Jawaban** untuk mengecek
5. Klik lagi tombol untuk menyembunyikan jawaban sebelum sesi berikutnya

---

## Cara Membaca Jam (Ringkasan Materi)

| Jarum | Warna | Fungsi | Rumus |
|-------|-------|--------|-------|
| Pendek | Hitam | Penunjuk JAM | Baca langsung angka yang ditunjuk |
| Panjang | Biru | Penunjuk MENIT | Angka × 5 = menit |
| Kurus | Merah | Penunjuk DETIK | Angka × 5 = detik |

**Contoh:** Jarum pendek di antara 3 dan 4, jarum panjang di angka 6
→ Pukul **03:30** (tiga lewat tiga puluh menit)
